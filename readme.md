**BuildFazz**<br />
<br />
_BuildFazz_ is the docker image builder.<br />
<br />
How to install : <br />
`go get github.com/payfazz/buildfazz/cmd/buildfazz`<br />
<br />
How to use : <br />
`buildfazz COMMAND [OPTIONS] {project-name}:"[project-tag]`<br />

Command exist :
- build<br />

Options exist :<br />
- -p         Specify path for project path
<br /><br />
See the help in:<br />
`buildfazz --help`<br /><br />
